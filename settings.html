PKM Data Entry Assistant

CNIC Configuration

Configure CNIC numbers for auto-filling on PKM website forms.

Press this key on the PKM website to auto-fill CNIC fields only.

Email Configuration

Configure email address for OTP notifications (same email will be used for all fields).

Press this key on the PKM website to auto-fill Email fields only.

Send OTP Settings

Press this key on the PKM website to send OTPs for all fields (constable, moharar, frontdesk).

OTP Configuration

Configure OTP numbers for auto-filling on PKM website forms.

Press this key on the PKM website to auto-fill OTP fields only.

Verify OTP Settings

Press this key on the PKM website to verify OTPs for all fields (constable, moharar, frontdesk).

Criminal Record Setting

When enabled, the extension will automatically select "No" in the Criminal Record dropdown.

Press this key on the PKM website to manually select "No" in the Criminal Record dropdown.

File Selection Setting

When enabled, the extension will scroll to the file input field and highlight it for easy access.

Press this key on the PKM website to scroll to the file input field and highlight it.

Save Record Setting

Press this key on the PKM website to click the Save Record button.

Combined Auto-Fill Settings

When enabled, the extension will automatically fill fields on the PKM website.

Press this key on the PKM website to auto-fill ALL fields (CNIC, OTP, Email) at once.

Data Fetching Settings

When enabled, the extension will simulate realistic typing behavior to trigger the website's data fetching mechanism. This should help when the website doesn't automatically fetch data after CNIC is filled.
